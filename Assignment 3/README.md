## Project Summary

NamLog Freight Tracker is a desktop freight tracking application built for the Advanced Programming assignment. The project demonstrates a Tkinter GUI connected to a Flask REST API, SQLAlchemy ORM models, SQLite persistence, and a background manifest worker.
